Exercise 1: Plot a Function
In this exercise, you will plot the function $$y = f(x) = -(x-5)^2.$$ Do this by defining an array \(X\) of \(n\) equally spaced values between 0 and 10. Begin with \(n = 100\) points in the array. For each element \(x\) in the array \(X\) (in mathematical notation we write, for each \(x \in X\)), compute the value of \(y\) given in the equation above. Finally, plot the pairs (x,y).
